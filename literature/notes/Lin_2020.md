## Key Concepts

### Spectrum of RL Agents
1. **Multi-armed bandits**: No context, pure exploration/exploitation
2. **Contextual bandits**: Use opponent's last action
3. **Full RL**: Model full history and state

### Human Behavior Mimicry
- Calibrate to human experimental data
- Match cooperation rates
- Reproduce reciprocity patterns

### Tournament Evaluation
- Multiple agent types compete
- Round-robin tournament
- Evolutionary fitness comparison
